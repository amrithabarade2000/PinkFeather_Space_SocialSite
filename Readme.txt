PinkFeather_Space_SocialSite is a social networking site where users can have their own personal accounts, create new groups, join already existing groups, post articles, etc... This is a semi-clone of Facebook with a galaxy-theme.

The following technologies were used in this project:

Programming Languages:
•	HTML
•	CSS
•	JavaScript
•	Python

Framework:
•	Django

Design Templates:
•	Bootstrap
•	Font awesome
•	Medium Editor
•	Google Fonts
